# 心电图机工作站

#### Description
6导、12导等多导联心电图机工作站管理系统源码，是和心电图机通讯的上位机pc软件，功能包含：网络通信（串口/网口），电子病历，设备数据采集、存储，诊断分析 ，心电图绘制，心电曲线转图片，心电图打印，多语言支持，DICOM 通讯协议 等等 ，国内上市医疗设备公司软件，希望能帮到有需要的朋友，学习参考少走弯路。
